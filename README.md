## Airline Reservation System

### To run this project

1. Clone using : git clone
2. Open a terminal at project root directory
3. To Compile
   - cd src
   - javac -cp ".;../lib/jcalendar-1.4.jar" -d ../bin com/files/\*.java
4. To Run
   - cd ..
   - cd bin
   - java -cp ".;../lib/jcalendar-1.4.jar" com.files.Main
